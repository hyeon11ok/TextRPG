# 🛡️ TextRPG - 스파르타 C# 과제

클래스의 구조를 신경쓰며 작성하였고 각 클래스의 역할과 그 역할에 맞는 기능을 고려했습니다.

---

## ✅ 필수 기능 구현

- 게임 시작 화면
- 상태 보기
- 인벤토리
- 장착 관리
- 상점
- 아이템 구매

---

## 🚀 도전 기능 구현

| 번호 | 기능 설명                     |
|------|------------------------------|
| 1    | 아이템 클래스 구현            |
| 2    | 아이템 정보를 Dictionary로 관리 |
| 3    | 아이템 추가 기능 구현         |
| 4    | 휴식 기능 추가                |
| 5    | 판매 기능 추가                |
| 6    | 아이템 장착 기능 개선         |
